Welcome to the TeamHuddle Web App
---------------------------------

Environment:
Ruby 2.0.0
Rails 4.1

I suggest using the RubyInstaller from http://rubyinstaller.org/downloads/
then running

###Setup Instructions:###


` $ bundle update `

` $ rake db:migrate `

` $ rake db:fixtures:load `

` $ rake bower:install `

` $ rails s -p 9090 ` //Run this server on Port 9090
